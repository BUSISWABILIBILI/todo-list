# Todo Project

A small todo app built one step at a time with HTML, CSS, and JavaScript.

## How to Open

Open `index.html` in a browser.

## Features

- Add tasks
- Mark tasks complete
- Delete tasks
- Save tasks in localStorage
- Filter all, active, and completed tasks
- Clear completed tasks
- Edit task names
